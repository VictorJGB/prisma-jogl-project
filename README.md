## Getting Started

### Prerequisites

    - Java Development Kit 11.0+
    - Gluegen and JOGL libraries installed

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VictorJGB/prisma-jogl-project
   ```
2. Open the n1-project folder
3. Run the `Main.java` file
